Team Explorer Everywhere for Visual Studio™ Team Foundation Server

You can use Team Explorer Everywhere in conjunction with Microsoft Team Foundation Server to improve the predictability of your development processes. Team Explorer Everywhere includes both the plug-in for Eclipse and the Cross-platform Command-line Client for Team Foundation Server. For more information on using these products please see the online guidance. For additional help and information visit the MSDN Forums.
