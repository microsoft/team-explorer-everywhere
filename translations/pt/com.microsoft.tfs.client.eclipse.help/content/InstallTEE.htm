Instalando o Cliente da Linha de Comando e o Plug-in do Team Explorer Everywhere

Você pode usar o Team Explorer Everywhere e o Visual Studio Team Foundation Server como parte dos seus processos de desenvolvimento com o Eclipse para rastrear itens de trabalho, fornecer controle de versão e automatizar compilações. Você deve instalar o plug-in do Team Explorer Everywhere para Eclipse ou o Cliente de Linha de Comando de Plataforma Cruzada para o Team Foundation Server antes de poder usar esses recursos.

Neste tópico

  • Pré-requisitos
  • Instalar o plug-in do Team Explorer Everywhere para Eclipse
  • Instalar o cliente de linha de comando de plataforma híbrida para Team Foundation Server

Pré-requisitos

Você deve instalar uma versão com suporte do Eclipse, a qual pode ser obtida no seguinte site: Home page de Eclipse.org..

Requisitos do sistema operacional:

  • Windows 10 (x86 and x64)
  • Windows 8 (x86, x64)
  • Windows 7 (x86, x64)
  • Linux com GLIBC 2.3+ (x86, x86_64)
  • Mac OS X 10.8+ (apenas Intel)

Requisitos do IDE (ambiente de desenvolvimento integrado):

  • IDEs que são baseados em Eclipse 4.2+ em Linux, Mac OS X, ou Windows.

Requisitos do Java Runtime:

  • Oracle Java™ 1.7+ ou IBM Java™ 1.7+ em Linux
  • Apple Java™ 1.7+ em Mac OS X
  • Oracle Java™ 1.7+ ou IBM Java™ 1.7+ em Windows

Requisitos do Team Foundation Server:

  • Visual Studio Team Services
  • Visual Studio Team Foundation Server 2017
  • Visual Studio Team Foundation Server 2015
  • Visual Studio Team Foundation Server 2013
  • Visual Studio Team Foundation Server 2012

Instalar o plug-in do Team Explorer Everywhere para Eclipse

É preciso instalar o plug-in para poder utilizá-lo em processos de desenvolvimento. Você pode instalar no Eclipse diretamente pelo site de atualização ou pelo arquivo morto de sites de atualização do Eclipse.

Para instalar o plug-in por meio do site de atualização

  1. Abra o Eclipse.
  2. No menu Ajuda, escolha Instalar novo software.
  3. A caixa de diálogo Instalar aparece.
  4. Escolha o botão Adicionar.
  5. A caixa de diálogo Adicionar repositório aparece.
  6. Para o nome, insira "Team Explorer Everywhere"
  7. Especifique o local do site atualizado: http://dl.microsoft.com/eclipse/tfs
  8. Escolha o botão OK.
  9. Na lista de recursos na caixa de diálogo Instalar, selecione a caixa de seleção que corresponde ao Plug-in do Team Explorer Everywhere para Eclipse. Se você não ver essa opção, use o menu suspenso de Trabalhar com, localize a URL de site de atualização inserida na lista e selecione-a, em seguida selecione a caixa de seleção ao lado do plug-in mencionado acima.
  10. Escolha o botão Avançar duas vezes.
  11. Se você concordar com os Termos de Licença para Software Microsoft, aceite-os e clique no botão Concluir.
  12. Reinicie o Eclipse quando for solicitado.

Para instalar o plug-in pelo arquivo morto de sites de atualização

    Observação
    O mecanismo do site de atualização padrão acima é o método de instalação preferencial, porém, exige uma conexão com a Internet e instalará a versão mais recente do plug-in da Microsoft. Pode ser mais adequado seguir o método se o computador de instalação não estiver diretamente conectado à Internet ou se a organização preferir que uma versão específica do plug-in seja instalada por todos.

  1. Use o link nessa página para baixar o arquivo TFSEclipsePlugin-UpdateSiteArchive-14.0.4.zip.
  2. Abra o Eclipse.
  3. No menu Ajuda, escolha Instalar novo software.
  4. A caixa de diálogo Instalar aparece.
  5. Escolha o botão Adicionar.
  6. A caixa de diálogo Adicionar repositório aparece.
  7. Para o nome, insira "Arquivo de atualização do Local Team Explorer Everywhere".
  8. Escolha o botão Arquivo morto.
  9. A caixa de diálogo Arquivo morto do repositório aparece.
  10. Localize e selecione o arquivo "TFSEclipsePlugin-UpdateSiteArchive-14.0.4.zip" baixado como o nome do arquivo.
  11. Escolha o botão Abrir.
  12. Escolha o botão OK.
  13. Nessa lista de recursos na caixa de diálogo Instalar, selecione a caixa de seleção que corresponde ao Plug-in do Team Explorer Everywhere para Eclipse
  14. Escolha o botão Avançar duas vezes.
  15. Se você concordar com os Termos de Licença para Software Microsoft, aceite-os e clique no botão Concluir.
  16. Reinicie o Eclipse quando for solicitado.

Instalar o cliente de linha de comando de plataforma cruzada para Team Foundation Server

É possível realizar várias operações de controle de versão no cliente de linha de comando de plataforma cruzada para Team Foundation Server. Antes de poder usar esse cliente, você precisa instalá-lo e configurar o caminho do sistema ou shell para incluir a pasta de instalação.

Para instalar o cliente de linha de comando de plataforma cruzada para Team Foundation Server

  1. Use o link nesta página para baixar TEE-CLC-14.0.4.zip.
  2. Descompacte o arquivo morto (arquivo TEE-CLC-14.0.4.zip) que contém o cliente.
  3. Configure o caminho do sistema ou shell para incluir a pasta na qual o arquivo morto foi descompactado.
  4. Para verificar se o cliente está funcionando, digite tf em um prompt de comando e pressione a tecla ENTER.
Se o cliente estiver instalado corretamente, aparecerá uma saída, começando com “Cliente da linha de comando do Microsoft Team Explorer Everywhere (versão 14.0.4)".

Para obter informações sobre qualquer um dos comandos, digite tf help Comando, em que Comando é o nome do comando sobre o qual você deseja obter informações. Por exemplo, você poderia digitar tf help checkin. Também é possível digitar tf command /help ou tf command -help para obter ajuda.

Observação
É possível controlar o comportamento do cliente da linha de comando configurando as variáveis de ambiente no shell, como especificar um Proxy HTTP a ser usado, habilitar cache automático das suas credenciais ou controlar a ferramenta comparar/mesclar. Para obter mais informações sobre as variáveis de ambiente a definir, consulte Definir variáveis de ambiente tópico na MSDN.
