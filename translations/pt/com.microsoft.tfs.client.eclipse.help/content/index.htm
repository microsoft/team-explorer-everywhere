Team Explorer Everywhere para Visual Studio Team Foundation Server

Você pode usar o Team Explorer Everywhere em conjunto com o Microsoft Team Foundation Server para aumentar a capacidade de previsão de seus processos de desenvolvimento. O Team Explorer Everywhere inclui o plug-in do Eclipse e o Cliente da Linha de Comando de Plataforma Cruzada do Team Foundation Server. Para obter mais informações sobre como usar esses produtos, consulte as orientações online. Para obter ajuda e informações adicionais, visite os Fóruns do MSDN.
